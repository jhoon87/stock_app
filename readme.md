# 주식 데이터 분석 대시보드

이 프로젝트는 Streamlit을 사용하여 주식 데이터를 분석하고 시각화하는 대시보드입니다.

## 기능
- 주식 코드를 입력하여 주가 데이터를 가져옵니다.
- 이동평균선, 볼린저 밴드, RSI, MACD, 스토캐스틱 오실레이터를 계산하고 시각화합니다.

## 사용 방법
1. Streamlit Sharing에서 앱을 실행합니다.
2. 주식 코드와 날짜 범위를 입력합니다.
3. 그래프 설정을 선택하여 데이터를 분석합니다.

## 필요한 패키지
- `streamlit`
- `yfinance`
- `matplotlib`
- `pandas`
- `numpy`
- `fonttools`
